Cybersecurity

Security-first
development

I don't treat security as an afterthought. Every application I build incorporates these principles from the design phase onward.

Secure Authentication

Robust auth flows with hashed passwords, session management, and brute-force protection.

Input Validation

Sanitising all user inputs to prevent SQL injection, XSS, and other injection-based attacks.

Secure Data Handling

Encrypting sensitive data at rest and in transit, with least-privilege access across all layers.

Threat Modelling

Identifying and analysing threats during the design phase, before a single line is written.

Security Auditing

Reviewing code for vulnerabilities and applying patches proactively to reduce attack surface.

OWASP Awareness

Building with the OWASP Top 10 in mind — guarding against the most critical web security risks.
